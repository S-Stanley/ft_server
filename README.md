# FT-SERVER

### BUILD
```
docker build . -t server
```

### RUN

```shell
docker run -d server
```

### HELP
If you have access problemes

```shell
sudo chmod 777 /var/run/docker.sock
```

If you have problemes to download images

```shell
docker login
```
