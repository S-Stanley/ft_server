# FT-SERVER

### BUILD
```
docker build . -t server
```

### RUN

```shell
docker run -d -p 127.0.0.1:80:80 server
```

### HELP
Useful cmd

```shell
sudo chmod 777 /var/run/docker.sock
```

```shell
docker login
```
